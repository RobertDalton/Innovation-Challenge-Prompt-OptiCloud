# Innovation Challenge Prompt OptiCloud Frontend

This is the frontend repository for the Innovation Challenge Prompt OptiCloud project. This project aims to provide an innovative solution for cloud optimization.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)

## Installation

To get started with the frontend, follow these steps:

1. Clone the repository:

```sh
git clone https://github.com/yourusername/Innovation-Challenge-Prompt-OptiCloud.git
```

2. Navigate to the frontend directory:

```sh
cd Innovation-Challenge-Prompt-OptiCloud/frontend
```

3. Install the dependencies:

```sh
pnpm install
```

## Usage

To start the development server, run:

```sh
pnpm dev
```

This will launch the application in your default web browser. The development server will reload automatically if you make any changes to the code.
